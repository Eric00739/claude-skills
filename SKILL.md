# 数学教学视频生成器 (Math Video Generator)

## 核心功能

将数学题目自动转换为高质量教学视频，包含完整讲解流程：

**8步工作流**：
1. 数学建模分析
2. HTML+SVG可视化讲解
3. 分镜脚本设计
4. TTS配音生成（自然语音）
5. 音频验证与时长更新
6. Manim脚手架生成
7. 完整动画代码实现
8. 视频渲染与输出

## 触发场景

- ✅ 用户粘贴数学题图片
- ✅ 用户提供数学题文本
- ✅ 用户要求"生成数学视频"
- ✅ 用户要求"数学讲解"
- ✅ 用户要求"教学动画"
- ✅ 用户要求 HTML 可视化讲解

## 输入格式

**文本题目**：
```
题目：[完整的数学题目描述]
```

**图片题目**：
```
[图片路径] 或粘贴图片后自动识别
```

## 输出文件

每个项目生成以下文件：

| 文件 | 说明 |
|------|------|
| `math_analysis.md` | 数学建模和推导过程 |
| `数学_{日期}_{题目}.html` | HTML+SVG可视化讲解文档 |
| `{日期}_{题目}_分镜.md` | 详细分镜脚本（含音频时长） |
| `audio/*.wav` (10个文件) | TTS配音文件 |
| `audio/audio_info.json` | 音频时长信息 |
| `script.py` | Manim动画代码 |
| `最终视频.mp4` | 1080p60高清视频 |

## 依赖技能

- **tutor** - 提供模板和脚本支持
- **opc-cli** - Qwen-TTS自然语音生成（必需）
- **manim** - 数学动画渲染引擎

## 工作流程详解

### 步骤 1：数学建模分析

**目标**：建立正确的数学计算方法和流程

**输出文件**：`math_analysis.md`

**内容结构**：
```markdown
## 数学事实分析

### 已知条件
- 条件1：...
- 条件2：...

### 推导的事实
1. **事实名称**: 描述
   - 计算过程: ...
   - 数学表达: ...

### 图形构建方法
- 点的坐标: ...
- 边的关系: ...

### 需要证明的结论
- 结论1: ...
```

**关键原则**：
- ❌ **禁止使用坐标法求解**（几何题）
- ✅ 使用几何推理：等积变换、相似三角形、勾股定理、向量叉积等
- ✅ 所有证明题的结论可作为已确立的事实使用

---

### 步骤 2：HTML+SVG 可视化

**目标**：用 HTML+SVG 画出图形，展示画图过程和解答流程

**输出文件**：`数学_{日期}_{题目简述}.html`

**包含内容**：
- 题目陈述
- SVG 图形（展示**画图过程**）
- 分步解答
- 关键要素标注

**SVG 设计规范**：
- 展示逐步画图过程（如：先画三角形 → 再画圆 → 标注点）
- 使用不同颜色标注关键部分
- 简洁清晰的视觉设计

---

### 步骤 3：分镜脚本设计

**目标**：定义视频结构，不限制幕数

**输出文件**：`{日期}_{题目}_分镜.md`

**分镜结构**：
```markdown
## 分镜设计

### 第1幕：{幕名}
**画面**: ...
**字幕**: ...（简洁，≤20字）
**读白**: ...（详细，口语化）
**动画**: ...
**目的**: ...

---

## 音频生成清单（步骤4填写）

| 幕号 | 文件名 | 读白文本 | 时长 | 说话人 | 情感 |
|------|--------|----------|------|--------|------|
| 1 | audio_001_{幕名}.wav | "读白文本" | | Serena | 温暖自然 |
```

**字幕退场约定**：
- 使用 `→` 箭头符号表示退场
- 或使用 `退场:` / `淡出:` 显式标记
- 或使用 `持续X秒` 指定显示时长

**关键规范**：
- 幕号从1开始连续编号
- 文件名格式：`audio_{三位幕号}_{幕名}.wav`
- **时长列为空**，由步骤5填充
- 根据内容需要决定幕数（通常10-15幕）

## 常见问题

### Q1: 声音听起来很自然吗？

**答案**：是的！默认使用 Qwen-TTS Serena 音色

- ✅ 温暖温柔的女声
- ✅ 像真人老师讲解
- ✅ 可自定义情绪指令
- ✅ 无机械感

**标准配置**：
```bash
--engine qwen --speaker Serena \
--instruct "用温暖自然的语气讲解，像老师一样亲切"
```

### Q2: 几何题如何避免坐标法？

**答案**：使用纯几何推理
- 等积变换（同底等高、等底同高）
- 相似三角形比例关系
- 勾股定理边长关系
- 向量叉积（纯几何意义）

**示例**：
❌ 错误：设B=(0,0), C=(5,0), A=(1.8, 2.4)...
✅ 正确：由勾股定理 BC = √(AB² + AC²)，正方形性质 BE = BC = 5

### Q3: 如何确保音频同步？

**答案**：
- 动画时长 >= 音频时长（画面等待音频）
- 使用 `duration` 参数控制
- 添加 `self.wait()` 延长时间

### Q4: 如何避免 LaTeX？

**答案**：全部使用 Text
```python
# ❌ 需要LaTeX
MathTex(r"BC^2 = AB^2 + AC^2")

# ✅ 无需LaTeX
Text("BC² = AB² + AC²", font_size=36)
```

**常用替换**：
- 上标：`x^2` → `x²`（Unicode）
- 分数：`\frac{1}{2}` → `½` 或 `1/2`
- 度数：`90^\circ` → `90°`
- 根号：`\sqrt{25}` → `√25`
