# Math Video Generator Skill

## 简介

这是一个自动生成数学教学视频的 Claude skill，基于完整的 tutor 工作流实现。

## 功能特点

✅ **8步完整工作流**
1. 数学建模分析
2. HTML+SVG可视化讲解
3. 分镜脚本设计
4. TTS配音生成（支持自然语音）
5. 音频验证与时长更新
6. Manim脚手架生成
7. 完整动画代码实现
8. 视频渲染与输出

✅ **自然语音支持**
- Edge-TTS 快速生成
- Qwen-TTS 高质量自然语音（推荐）

✅ **自动化流程**
- 音画自动同步
- 时长自动验证
- 视频自动拷贝

## 使用方法

### 基本使用

直接向 Claude 提供数学题目：

```
题目：超市运来一批苹果，第一天卖出了总数的30%，第二天卖出了120千克，这时还剩下160千克。请问这批苹果原来有多少千克？
```

Claude 会自动：
1. 创建项目目录
2. 执行完整工作流
3. 生成所有文件
4. 渲染最终视频

### 高级使用

指定自然语音：

```
题目：[数学题目]
请使用 Qwen-TTS Serena 自然语音生成视频
```

## 输出文件

每个项目生成：
- `math_analysis.md` - 数学分析文档
- `数学_{日期}_{题目}.html` - HTML可视化讲解
- `{日期}_{题目}_分镜.md` - 详细分镜脚本
- `audio/*.wav` - 10个配音文件
- `script.py` - Manim动画代码
- `最终视频.mp4` - 1080p60高清视频

## 触发条件

Skill 会自动激活：
- 用户粘贴数学题图片
- 用户提供数学题文本
- 用户提到"数学视频"
- 用户提到"教学动画"
- 用户提到"数学讲解"

## 依赖

- **tutor skill** - 提供模板和脚本
- **opc-cli skill** - Qwen-TTS支持（可选）
- **manim** - 动画渲染引擎
