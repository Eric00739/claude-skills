## 数学事实分析

### 题目
超市运来一批苹果，第一天卖出了总数的 30%，第二天卖出了 120千克，这时还剩下 160千克。请问这批苹果原来有多少千克？

### 已知条件
- 条件1：第一天卖出了总数的 30%
- 条件2：第二天卖出了 120 千克
- 条件3：还剩下 160 千克

### 推导的事实

1. **建立方程模型**
   - 设未知数：设这批苹果原来有 x 千克
   - 第一天卖出量：30%x = 0.3x 千克
   - 第二天卖出量：120 千克（已知常数）
   - 剩余量：160 千克（已知常数）

2. **等量关系**
   - 数学表达：原有总量 - 第一天卖出 - 第二天卖出 = 剩余量
   - 方程形式：x - 0.3x - 120 = 160

3. **解方程过程**
   - 合同类项：0.7x - 120 = 160
   - 移项：0.7x = 280
   - 求解：x = 280 ÷ 0.7 = 400

4. **验证答案**
   - 原有总量：400 千克
   - 第一天卖出：400 × 30% = 120 千克 ✓
   - 第二天卖出：120 千克 ✓
   - 剩余：400 - 120 - 120 = 160 千克 ✓

### 需要证明的结论
- 这批苹果原来有 **400 千克**

### 关键数学概念
- 百分数的意义：30% = 0.3 = 30/100
- 一元一次方程：含有一个未知数，未知数的次数是1
- 等量关系：总量 = 各部分之和
